# 📁Network (네트워크)

백엔드 개발자 면접에서 필수적으로 물어보는 **HTTP 프로토콜**과 **REST API 설계**에 대한 내용을 정리한 폴더입니다.

## 📚 목차

### 1. [📄HTTP Protocol (Basic & Advanced)](http.md)
* **Method**: GET, POST, PUT, PATCH, DELETE 차이점
* **Idempotency**: 멱등성의 개념과 중요성
* **Status Code**: 2xx, 4xx, 5xx 주요 코드
* **Header**: 캐싱(ETag), 인증(Authorization)
* **Interview Topics**: Stateless, Keep-Alive, HTTPS 원리

### 2. [📄REST API 설계 가이드](REST_API.md)
* **Design Principles**: 자원 중심 설계, 행위의 명시
* **HATEOAS**: 개념과 예시
* **RMM**: Richardson Maturity Model (REST 성숙도 모델 0~3단계)
* **Self-descriptive Message**: 자체 표현 구조
